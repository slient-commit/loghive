How it works

Running in 3 steps

No managed infrastructure, no monthly bill. Your logs stay where you host them.

01
Deploy with Docker Compose

Add your database credentials to .env, run docker compose up -d. The API and frontend start in under a minute.

02
Create an app & API key

Register, create your first application in the dashboard, and generate an API key. Takes 30 seconds.

03
Install an SDK & start logging

Two lines of setup in Node.js, Python, or .NET. Logs appear in real time — searchable, filterable, exportable.

SDKs

2 lines to start logging

Official, maintained libraries for the platforms you use.

Node.js
npm install loghive-sdk
Python
pip install loghive-sdk
.NET (C# / .NET Standard 2.0)
Add LogHive.dll reference 
// npm install loghive-sdk
const LogHive = require('loghive-sdk');

const logger = new LogHive({
  apiKey:   'lh_your_api_key',
  endpoint: 'https://logs.yourco.com',
});

await logger.info('User signed in', {
  tags:     ['auth'],
  metadata: { userId: 42 },
});

logger.queue('warn', 'Slow query', { metadata: { ms: 1842 } });
await logger.shutdown();
# pip install loghive-sdk
from loghive import LogHive

logger = LogHive(
    api_key  = "lh_your_api_key",
    endpoint = "https://logs.yourco.com",
)

logger.info("User signed in",
    tags=["auth"], metadata={"user_id": 42})

with LogHive(api_key="...", endpoint="...") as log:
    log.fatal("Unhandled exception", tags=["critical"])
using LogHive;

var logger = new LogHiveClient(
    "lh_your_api_key",
    "https://logs.yourco.com"
);

await logger.InfoAsync("App started");
await logger.ErrorAsync("Payment failed",
    tags: new[] { "payments" });

logger.Queue("warn", "High memory usage");
await logger.ShutdownAsync();
